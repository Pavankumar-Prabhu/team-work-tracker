# Workflow Document

This document outlines the WBR workflow for task management from Monday to Friday.

## Workflow Details
- **Workdays:** Monday to Friday
- **Carry Forward:** Tasks that are not completed on Friday will carry forward to Monday, skipping weekends.

## Tasks with Carry Forward Logic
- Tasks due on Friday will automatically shift to Monday if not completed.
- Ensure that weekend is not included in the carry forward.
